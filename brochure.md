# WordCoach 앱 브로슈어

최종 업데이트: 2026년 5월 3일

## 한 줄 소개

WordCoach는 외우고 싶은 단어를 직접 등록하고, 복습 큐와 알림, 위젯으로 매일 다시 만나게 해주는 iOS 단어 학습 앱입니다.

## 제품 개요

WordCoach는 사용자가 등록한 단어를 단순히 저장하는 데서 끝내지 않습니다. 단어, 뜻, 예문, 태그, 메모를 기기 안에 저장하고, 플래시카드와 객관식 퀴즈, 로컬 알림, 홈 화면 위젯을 통해 사용자가 자연스럽게 복습 루틴으로 돌아오게 만듭니다.

앱은 서버 없이 동작합니다. 단어장과 복습 기록은 SwiftData로 기기 안에 저장되고, 학습 알림은 iOS 로컬 알림으로 예약되며, 위젯은 App Group 저장소를 통해 오늘의 단어 스냅샷을 읽습니다.

## 핵심 기능

- 단어 등록, 수정, 삭제
- 뜻, 예문, 태그, 메모 저장
- 플래시카드 복습
- 객관식 퀴즈
- 정답률, 복습 횟수, 연속 정답 기록
- 하루 3번 동기부여 로컬 알림
- 오늘 등록한 단어를 보여주는 홈 화면 위젯
- Pro 사용자를 위한 스마트 복습 위젯
- 광고 기반 무료 사용과 StoreKit 2 기반 Pro 구매

## 무료 사용자 경험

무료 사용자는 단어장, 복습, 퀴즈, 기본 위젯을 사용할 수 있습니다. 하루 10개까지 단어를 등록할 수 있고, 리워드 광고를 보면 당일 단어 등록 가능 수가 5개 늘어납니다.

광고는 학습 흐름을 과하게 끊지 않도록 홈/복습 화면 상단 인라인 배너와 일정 활동 이후의 전면 광고로 구성했습니다.

## WordCoach Pro

WordCoach Pro는 학습 흐름을 더 빠르고 조용하게 만드는 유료 기능입니다.

- 광고 제거
- 일괄 단어 추가
- 저장 후 계속 입력
- 약한 단어를 우선하는 집중 복습
- 중간/큰 스마트 위젯
- 하루 단어 등록 제한 제거

추천 가격:

- 월간 Pro: 월 2,900원
- 평생 Pro: 19,000원

## 개인정보와 광고 방향

WordCoach는 첫 출시에서 App Tracking Transparency 권한을 요청하지 않습니다. 광고 요청 전 Google User Messaging Platform으로 필요한 동의 상태를 확인하고, Google Mobile Ads 요청에는 비개인화 광고 설정을 적용합니다.

사용자가 등록한 단어와 복습 기록은 자체 서버로 전송하지 않습니다. WordCoach Pro를 활성화하면 앱 화면에서 광고 요청을 보내지 않습니다.

## App Store 소개 문구 초안

외우고 싶은 단어를 등록해두면 WordCoach가 다시 만날 타이밍을 만들어줍니다. 플래시카드, 퀴즈, 로컬 알림, 홈 화면 위젯으로 단어를 매일 조금씩 복습하세요.

단어와 뜻, 예문, 태그, 메모를 직접 관리하고, 정답률과 복습 기록을 보며 나만의 단어장을 키울 수 있습니다. 서버 없이 기기 안에서 동작하므로 학습 데이터는 내 손 안에 머뭅니다.

WordCoach Pro를 사용하면 광고 없이 학습하고, 단어를 한 번에 붙여넣고, 약한 단어를 우선하는 집중 복습과 스마트 위젯을 사용할 수 있습니다.
