서버 없는 iOS 단어 학습 코치

외우고 싶은 단어를 오늘의 습관으로.

WordCoach는 사용자가 직접 등록한 단어를 복습 큐, 퀴즈, 로컬 알림, 홈 화면 위젯으로 자연스럽게 다시 만나게 해주는 SwiftUI 단어 학습 앱입니다.

SwiftUI SwiftData WidgetKit AdMob + UMP StoreKit 2

Product

단어 등록에서 복습까지, 사용자가 매일 돌아오게 만드는 작은 학습 루프.

01 내 단어장

외우고 싶은 단어, 뜻, 예문, 태그, 메모를 직접 등록하고 언제든 수정·삭제할 수 있습니다.

02 복습과 퀴즈

플래시카드와 객관식 퀴즈로 단어를 다시 만나고, 정답률과 복습 기록으로 다음 학습을 조정합니다.

03 알림과 위젯

하루 3번 로컬 알림과 홈 화면 위젯이 오늘 등록한 단어와 복습할 단어를 계속 떠올리게 합니다.

Experience

학습 앱답게 가볍고 빠르게. 광고와 Pro 기능은 흐름을 방해하지 않게.

무료 사용자

  • 하루 10개 단어 등록
  • 복습, 퀴즈, 기본 위젯 사용
  • 상단 인라인 배너, 일정 횟수 후 전면 광고
  • 리워드 광고 시 당일 단어 등록 5개 추가

WordCoach Pro

  • 광고 제거
  • 일괄 단어 추가와 연속 입력
  • 약한 단어를 우선하는 집중 복습
  • 중간/큰 스마트 위젯

Privacy

서버 없이 동작하고, 첫 출시는 ATT 없이 비개인화 광고 방향으로 설계했습니다.

기기 내 저장

단어장과 복습 기록은 SwiftData로 기기 안에 저장됩니다. 자체 서버로 학습 데이터를 전송하지 않습니다.

로컬 알림

동기부여 알림은 UserNotifications 기반 로컬 알림으로 예약됩니다. 푸시 서버가 필요 없습니다.

광고 동의

AdMob 광고 요청 전 Google UMP 동의를 확인하고, 첫 출시에서는 비개인화 광고 요청을 사용합니다.
